1.1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web interactivas y desarrollo de aplicaciones móviles.

Una página web interactiva es aquella que incorpora efectos (como texto que aparece y desaparece), animaciones, acciones que se activan al pulsar botones y ventanas con mensajes de aviso al usuario.

Técnicamente, JavaScript es un lenguaje de programación interpretado, por lo que no es necesario compilar los programas para ejecutarlos. En otras palabras, los programas escritos con JavaScript se pueden probar directamente en cualquier navegador sin necesidad de procesos intermedios.

Nota: Es de suma importancia dejar muy en claro que a pesar de su nombre, JavaScript no guarda ninguna relación directa con el lenguaje de programación Java. Legalmente, Java es una marca registrada originalmente por la empresa Sun Microsystems, y posteriormente adquirida por Oracle Corporation en 2009.

La sintaxis de JavaScript fue diseñada con mucha similitud a la del lenguaje C, aunque utiliza nombres y convenciones similares a las de Java. Sin embargo Java y JavaScript no están relacionados y tienen semánticas y propósitos diferentes.

Hoy en día existen gran cantidad de aplicaciones web tales como Youtube, Gmail, Facebook, Twitter, Flicker, Google Docs, etc, las cuales utilizan diversos lenguajes de programación del lado del servidor como Ruby, Java, PHP, .NET, Python, pero todos convergen en un único lenguaje, JavaScript que se ha convertido en una pieza clave al momento de desarrollar aplicaciones web altamente interactivas y de gran rendimiento, tanto en la web (dentro del navegador) como en dispositivos de nueva generación como Smart TVs, Smarthphones, Tablets, Drones, etc.

Por su simplicidad de sintaxis, compatibilidad con multiples tecnologías y plataformas JavaScript se ha convertido en una verdadera navaja suiza para los desarrolladores de software más exigentes y sin duda un lenguaje que todo desarrollador de sitios web, aplicaciones web o móviles debe conocer.
